# FraudShield

FraudShield is a microservices-based fraud detection system that processes transactions, identifies suspicious activity, and triggers real-time alerts.

## 🚀 Tech Stack

* Java, Spring Boot
* Maven (multi-module)
* OpenAPI (contract-first)
* Docker (planned)
* CI/CD (GitHub Actions)

## 🧩 Services

* Transaction Service
* Fraud Detection Service
* Alert Service
* API Contract Module

## ▶️ Run Locally

```bash
mvn clean install
mvn spring-boot:run
```

## 📌 Goal

Build a scalable, production-ready fraud detection system using modern microservices' architecture.
